# React Custom Hooks Demo (React 19 + Vite)

A ready-to-run sample app that teaches:
- Creating and using custom hooks
- Sharing logic (not state) across components
- Passing reactive values and handlers
- Effect cleanup and dependency arrays

## Quickstart
```bash
npm install
npm run dev
# open the local URL printed by Vite
```

## Build for production
```bash
npm run build
npm run preview
```
